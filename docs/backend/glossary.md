# Glossary

- **API**: 클라이언트(프론트)가 서버에 기능을 요청하는 인터페이스.
- **JWT**: 사용자 식별 정보를 담는 서명된 토큰.
- **Access Token**: 짧은 수명으로 API 인증에 직접 사용.
- **Refresh Token**: Access Token을 재발급받기 위한 긴 수명 토큰.
- **HttpOnly Cookie**: JavaScript로 읽을 수 없는 쿠키. XSS 리스크 완화에 유리.
- **CORS**: 다른 Origin 간 요청 허용 정책.
- **Flyway**: DB 스키마 변경을 버전 단위로 관리하는 마이그레이션 도구.
- **Argon2**: 비밀번호 해시 알고리즘.
- **OAuth2**: 외부 인증 제공자(Google 등)로 로그인하는 표준 프로토콜.
- **Security Filter Chain**: 요청이 컨트롤러에 도달하기 전 인증/인가를 처리하는 체인.
